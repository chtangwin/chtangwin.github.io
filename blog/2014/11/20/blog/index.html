github blog setup

Nov 20, 2014

Now I have decided to use github + pelican for personal blog. The plan is to integrate with ipython notebook so all my math/stats notes can be displayed easily.

Steps to Setup

  1. Follow the instructions from How to setup Github User Page with Pelican, up to the point where first post is done and local devserver is up. The only difference is I am using main branch ("master") for all edits.

  2. The first theme to use is from python4oceanographers.

Workflow

Once the basic setup is completed, use the following steps to add blog posts. 1. activate myblog 2. add new entries in "content/", including ipython notebooks, images, code, etc. 3. test locally 4. pushlish 5. deactivate myblog
